# adventure_works_project
### Project Title
Adventure Works Analytics Dashboard

### Short Description
Designed an end-to-end analytics dashboard to analyze global sales performance and uncover key business drivers for the Adventure Works manufacturing company.

### Tech Stack
Excel - Initial data profiling, missing value handling & KPI calculations
MySQL - Data modeling using joins & queries to combine production, sales & customer data
Power BI / Tableau - Modern dashboards with drill-down analysis for management decision-making

### Highlights
Built interactive visual dashboards for revenue, returns, and regional sales monitoring
Analyzed top-performing products which contributed ~65% of total company revenue
Identified high return rates in Bike & Component categories indicating quality control concerns
Implemented customer segment analysis to focus on high-value buyer groups

### Business Impact
Helped leadership focus on profitable product lines and enhance customer satisfaction by addressing return trends
